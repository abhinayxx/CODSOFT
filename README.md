# CODSOFT - Python Internship Projects

This repository contains the Python projects completed as part of the **CodSoft Internship Program**. Each project is a beginner-friendly, console-based application focused on building strong fundamentals in Python - covering input handling, conditionals, loops, lists, dictionaries, and basic program flow.

---

## 1. Calculator

A simple command-line calculator that performs basic arithmetic operations.

**Features:**
- Takes two numbers as input from the user
- Supports four operations: Addition, Subtraction, Multiplication, and Division
- Menu-driven choice system (1-4)
- Handles division by zero gracefully with an error message
- Displays an "Invalid choice" message for unsupported menu options

**Concepts used:** input/output handling, conditionals (if-elif-else), type casting (float)

---

## 2. Rock Paper Scissor

A classic Rock-Paper-Scissors game where the user competes against the computer.

**Features:**
- User enters their choice (Rock, Paper, or Scissors)
- Computer randomly selects its choice using Python's `random` module
- Compares both choices and determines the winner based on standard game rules
- Handles invalid inputs from the user
- Runs in a continuous loop, asking the user if they want to play again
- Exits gracefully with a thank-you message when the user stops playing

**Concepts used:** `random` module, while loops, conditional logic, string comparison

---

## 3. To-Do List

A console-based To-Do List manager to keep track of daily tasks.

**Features:**
- Add new tasks to the list
- View all tasks along with their status (Completed / Pending)
- Mark a specific task as completed by selecting its number
- Delete a specific task from the list
- Exit the application
- Menu-driven interface that loops until the user chooses to exit
- Handles invalid task numbers and empty task lists with appropriate messages

**Concepts used:** lists, dictionaries, loops, enumerate(), exception handling (try-except)

---

## Tech Stack
- **Language:** Python 3
- **Modules used:** `random` (for Rock Paper Scissor)

## How to Run
1. Navigate to the desired project folder
2. Run the script using:
   ```bash
   python main.py
   ```
3. Follow the on-screen prompts
